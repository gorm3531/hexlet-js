# Установка зависимостей

Для установки всех зависимостей проекта выполните:

```bash
make install
```

## Линтинг кода

Для проверки стиля кода используется ESLint.

### Команды

- `make lint` — проверка кода на соответствие правилам ESLint.
- `make lint-fix` — автоматическая проверка и исправление ошибок стиля.
